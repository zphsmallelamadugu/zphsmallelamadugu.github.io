# ZPHS Mallelamadugu

Static website for **Zilla Parishad High School, Mallelamadugu** — Aswapuram Mandal, Bhadradri Kothagudem District, Telangana.

Live: https://zphsmallelamadugu.github.io

## Stack
Plain HTML + Tailwind (CDN) + a small bit of JS. No build step. Deployed via GitHub Pages from `main`.

## Local preview
Open `index.html` directly in a browser, or run any static server:

```
python3 -m http.server 8000
```

Then visit http://localhost:8000.
